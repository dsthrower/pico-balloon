# Pico Balloon Project (KK7OWL)

Working reference for a 20m WSPR pico balloon build using the Traquito Jetpack tracker.

## Contents

- **[references.md](references.md)** — Links (Traquito, how-to guides, suppliers),
  flight identifiers (callsign, channels), and key build specs.
- **[receive_station_setup.md](receive_station_setup.md)** — SDR receive/monitor
  station: hardware, signal chain, and bring-up runbook for watching WSPR.

## Quick facts

- **Callsign:** KK7OWL
- **Band:** 20m WSPR (dial 14.0956 MHz USB)
- **Reserved channels:** 290, 291, 292
- **Tracker:** Traquito Jetpack (Raspberry Pi Pico), solar powered
- **Antenna:** 30 AWG tinned-copper dipole, 4325 mm/leg (8650 mm tip-to-tip),
  resonant ~14.020 MHz, SWR 1.33 on bench
